# 京东商品数据分析可视化系统 python774

## 项目概述

本项目是一款基于Python和Django框架的商品数据分析可视化系统。通过使用Selenium爬虫技术，系统从京东网站获取商品销售数据，进而为用户提供商品信息的查询、分析和预测功能。

## 技术栈

- 后端：Python、Django框架
- 数据库：MySQL
- 前端：Vue框架
- 可视化：Echarts
- 预测模型：线性回归模型

## 功能模块

- 商品数据爬取
- 数据存储管理
- 商品信息查询
- 销售趋势分析与可视化
- 销量预测

## 系统角色

- 用户：查询商品信息，分析销售趋势，进行销量预测

## 运行环境

- 后端：Python 3.8+、Django 3.x
- 数据库：MySQL 5.7+
- 前端：Vue.js 2.x，现代浏览器

## 部署步骤

1. 安装后端依赖：Python环境搭建，安装Django和MySQL
2. 配置数据库：设置MySQL数据库及数据表
3. 部署后端服务：运行Django应用
4. 安装前端依赖：安装Node.js，使用Vue CLI搭建前端项目
5. 前后端联调：确保前后端通信正常
6. 部署前端：将前端项目打包并部署至Web服务器

## 核心亮点

- 大屏可视化：首页展示直观的数据可视化效果
- 灵活查询：用户可根据需求自定义查询商品信息
